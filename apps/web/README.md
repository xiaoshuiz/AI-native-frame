# 🌐 Web App (Frontend-First)

This is the primary frontend application for the repository.

## ⚡ Scripts

```bash
npm install
npm run dev
```

Additional scripts:

- `npm run lint`
- `npm run typecheck`
- `npm run build`
- `npm run preview`

## 🧭 Working model

- Use `specs/templates/` to define requirements first.
- Implement UI in this app after specs are aligned.
- Ensure `lint` and `build` pass before opening a PR.
